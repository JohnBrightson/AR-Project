# AR Image Target Application (Vuforia)

##  About the Project
This is an Augmented Reality application developed using **Unity** and **Vuforia Engine**.
The app uses **two image targets** to display AR content when scanned through the camera.

##  Image Targets
The following reference images are used as image targets:
- Image Target 1
- Image Target 2  
(Available in this repository)

##  APK
- Download the APK from the **Releases** section
- Install it on an Android device
- Allow camera permission
- Scan the provided image targets to view AR content

##  Tools & Technologies
- Unity
- Vuforia Engine
- Android (APK build)
